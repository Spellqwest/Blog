# Addictive Patterns

| **Type**             | **Notes**                                                                                                                                                                                                                                                                |
| -------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **Explore**          | - Random generation to make every run unique<br />**Come and try**<br />- Different starting spells<br />**Come back**<br />- Meta-progression (Roguelite-Elements)<br />- New milestone unlocks (e.g. new stages/map after defeating a boss)<br />- New types of spells |
| **Express / Create** | - Trying out different strategies (come and try / come back depending on the level strategy)<br />**Come back**<br />- Unlock specific spells or spell-types through meta-progression resources                                                                          |
| **Collaborate**      | **Bring friends**<br />- Sharing progress or tips with friends                                                                                                                                                                                                           |
| **Compete**          | **Come and try**<br />- Challenging combat<br />**Come back**<br />- (Maybe progressing)<br />- Late-game challenges<br />- Death counter / win-streak                                                                                                                   |
